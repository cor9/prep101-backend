# How to Check and Disable IPv4 - Use the Dashboard

## Simple Steps (No API, No Tokens)

**1. Open Supabase Dashboard:**
   - Go to: https://supabase.com/dashboard
   - Log in

**2. Select Your Project:**
   - Click on project: `Prep101` (or `eokqyijxubrmompozguh`)

**3. Go to Settings:**
   - Click "Settings" in the left sidebar

**4. Find Add-ons:**
   - Look for "Add-ons" or "Infrastructure" section
   - OR check "Billing" section

**5. Look for IPv4:**
   - Find "Dedicated IPv4" or "IPv4 Add-on"
   - See if it's enabled (toggle ON) or disabled (toggle OFF)

**6. If Enabled, Turn It Off:**
   - Click the toggle to turn it OFF
   - Confirm if asked
   - Done! Saves ~$4/month

---

## That's It!

- No API calls needed
- No tokens needed
- Just use the dashboard UI
- Your app will continue working (uses modern connections)
